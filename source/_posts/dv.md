# Data Visualization 1 : Intro 
## Clustering 聚类
### Definition  

+ **Input**: A database of n data objects in R and a certain distance function  
+ **Output**: Split the data objects into clusters so that objects in the same cluster are much closer to each other than those from other clusters

### k-partitioning Clustering
Given a set of n data objects $D = \{x_1,x_2,...,x_n\}$ in $R$ and a certain distance function $dist()$
#### Distance Function
+ $L_p-norm$
+ Euclidean Distance, $L_2-norm$
+ Manhattan Distance, $L_1-norm$
+ Weight : Capture importance of attributes
+ Scaling : Normalization of attributes to within the same range
